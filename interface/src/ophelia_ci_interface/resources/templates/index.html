Ophelia CI: Your Open-Source Git Server Solution

Ophelia CI empowers teams to host and manage their own Git repositories with ease.

This comprehensive solution provides:

A robust Git Server: A high-performance Linux daemon, written in Go, available in multiple formats (.deb, .snap, .rock) and via Docker, ensuring seamless integration into your infrastructure.
A versatile CLI Client: A command-line interface, also built in Go and distributed in the same formats, for efficient repository management.
A user-friendly Web Interface: A graphical web application, developed with Python and FastAPI, offering an intuitive experience, packaged in various formats (.deb, .snap, .rock, .AppImage, .flatpak) and through Docker.
Ophelia CI is designed for flexibility and control, allowing you to tailor your Git workflow to your team's specific needs.

Coming Soon: Integrated CI/CD

Our roadmap includes the development of a built-in Continuous Integration (CI) tool. Users will be able to define workflows using YAML configurations, automating tests and ensuring code quality directly within Ophelia CI.
